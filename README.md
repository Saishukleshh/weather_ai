# Techfiesta.app

A comprehensive agricultural management and monitoring application built with React (Frontend) and Node.js/Express (Backend).

## Features

- **Agro Monitoring**: Real-time monitoring of agricultural activities.
- **Crop Recommender**: AI-powered crop recommendation system.
- **Weather Dashboard**: Weather forecasting and analysis for farming decisions.
- **Farming Schedule**: Planning and scheduling farming tasks.
- **Risk Forecast**: Risk assessment for crops and weather-related issues.
- **User Map**: Interactive maps for farm management.
- **Grain Card**: Tracking and management of grain inventory.
- **Crop Data API**: API for accessing crop-related data.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Various React components for UI

### Backend
- Node.js
- Express.js
- Axios for HTTP requests
- Other dependencies as per package.json

## Installation

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Saishukleshh/weather_ai.git
   cd Techfiesta.app
   ```

2. Install backend dependencies:
   ```bash
   cd Backend
   npm install
   ```

3. Install frontend dependencies:
   ```bash
   cd ../Frontend
   npm install
   ```

4. Start the backend server:
   ```bash
   cd Backend
   npm start
   ```

5. Start the frontend development server:
   ```bash
   cd Frontend
   npm run dev
   ```

## Usage

- Access the frontend at `http://localhost:5173` (default Vite port)
- Backend API runs on `http://localhost:3000` (or as configured)
